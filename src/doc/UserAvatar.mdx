# UserAvatar

UserAvatar can be used to represent people or objects. It supports images or letters.
## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| isShowUsername | wheather show username | boolean | `false` |
| shortName | string of letters | string | - |
| username | string of name | string | - |
| type | can be set to default, primary, success, warning, danger | string | `primary` |


> Note: Accept all [Avatar Properties](https://ant.design/components/avatar/?locale=en-US#API)
