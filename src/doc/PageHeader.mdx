# PageHeader

Panel slides from screen edge with form.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| isLoading |whether the skeleton is visible or not.| boolean | - |
| leftActionControls\|rightActionControls | content of the left \| right controls|string\|ReactNode | - |
| popoverDropdownItems | content of the card | string\|ReactNode | - |
| popoverDropdownTrigger | whether floating PopoverDropdown by `click` or `hover` | string | `click` |
| title | the title for PageHeader. | string | - |
