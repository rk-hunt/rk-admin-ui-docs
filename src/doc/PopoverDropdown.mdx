# PopoverDropdown

The floating dow list popped by clicking or hovering.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| className |string of className | string | `no-arrow` |
| content | content of the card | string\|ReactNode | - |
| placement | The position of the PopoverDropdown relative to the target, which can be one of `top` `left` `right` `bottom` `topLeft` `topRight` `bottomLeft` `bottomRight` `leftTop` `leftBottom` `rightTop` `rightBottom` | string | `bottomLeft` |
| trigger | whether floating PopoverDropdown by `click` or `hover` | string | `hover` |
