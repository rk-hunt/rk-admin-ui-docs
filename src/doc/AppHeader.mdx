# AppHeader

The global header of web app.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| collapsed | If `true`, side menu cannot be opened or closed | boolean | `false` |
| menuFoldIcon | string of [icon](https://ant.design/components/icon/?locale=en-US#API) type | string | `menu-fold` |
| menuUnfoldIcon | string of [icon](https://ant.design/components/icon/?locale=en-US#API) type | string | `menu-fold` |
| notificationIcons | [NotificationIcon](/notification-icon) properties | ReactNode | - |
| onSearch |handler of search box | func | - |
| popoverDropdownContent | Content of the card | string\|ReactNode | - |
| popoverDropdownTrigger | whether floating PopoverDropdown by `click` or `hover` | string | `click` |
| searchPlaceholder |placeholder of search box | string | `Search` |
| searchPopoverDropdownTrigger | whether floating PopoverDropdown by `click` or `hover` | string | `click` |
| user | object of [UserAvartar](/user-avatar) properties | object | - |
