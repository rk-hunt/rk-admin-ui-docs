# ModalForm

When requiring users to interact with the application, but without jumping to a new page and interrupting the user's workflow, you can use Modal to create a new floating layer over the current page to get user feedback or display information.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| columns | whether to render input elment with 2 columns or more| number | `1` |
| modalProps |accept all [Antd Design Modal](https://ant.design/components/modal/?locale=en-US#API) properties | object | - |
| fields |array object of [fields properties](/dynamic-element#fields-properties) | array object | - |
