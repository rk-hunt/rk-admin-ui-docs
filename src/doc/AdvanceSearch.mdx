# AdvanceSearch

is a component which allows a user to specify additional requirements for a search.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| columns |array object of [columns properties](/editable-table#columns-properties) | number | `4` |
| fields |array object of [DynamicElement Properties](/dynamic-element#fields-properties) | boolean | - |
| firstTabName | Show text in TabPane's head | string | `General` |
| activeKey | Key of the active panel | string[]\|string | - |
| cancelButtonText | Text of the CANCEL button | string\|ReactNode | `Cancel` |
| searchButtonText | Text of the SEARCH button | string\|ReactNode | `Search` |
| onCancel | Specify a function that will be called when a user clicks cancel, close button on top right or CANCEL button | function(e) | - |
| onSearch | Specify a function that will be called when a user clicks button SEARCH | function(e) | - |
