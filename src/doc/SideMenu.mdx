# SideMenu
Menu list of Navigation.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| collapsed |to set the current status | boolean | - |
| defaultOpenKeys |array with the keys of default opened sub menus | (collapsed, type) => {} | - |
| isMobile |to set current status of screen | boolean | `false` |
| defaultSelectedKeys |array with the keys of default selected menu items | (collapsed, type) => {} | - |
| onCollapse |the callback function, executed by clicking the trigger or activating the responsive layout | (collapsed, type) => {} | - |
