# EditableTable

A table displays rows of data.

## Properties

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| columns |array object of [columns properties](/editable-table#columns-properties) | boolean | - |
| editable |whether table row is able to edit | boolean | `false` |
| onFieldChange | hanlder of column element blur | function(form, name, record, index) | - |

## columns Properties
| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| fields |array object of [DynamicElement Properties](/dynamic-element#fields-properties) | boolean | - |
| column | table properties, [read details](https://ant.design/components/table/#API) | boolean | `false` |
